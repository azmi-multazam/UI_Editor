Whats new ?


Nama : UI_Editor.apk
Version : 2.0

==== FITUR ====

- Relayout, buat meminimalisir pemakain file png supaya file aplikasi tetap kecil dan ringan

- Tap photo profil untuk mengambil photo profil.

- Tap tanda >>> untuk mengambil photo cover.

- Menu Setting, buat milih layout photo profile yg mau dipake, kotak apa bulet?.. terserah.. :)

- Opsi cover quality, sebagian device ada yg ga support cache dengan ukuran besar, opsi ini buat milih qualitas gambar cover high, medium atau low dan memilih ratio crop 3:2 atau 2:1. gunakan settingan Medium atau Low kalo loading lama atau FC waktu cropping cover.

- Custom Fonts, terserah agan suka font seperti apa.. , tinggal rename font agan dengan nama font berikut, jangan lupa extensinya kudu ttf

name.ttf - buat nama profil
quote.ttf - buat quote
carrier1.ttf - buat text1
carrier2.ttf - buat text2
carrier3.ttf - buat text3

taro di sdCard/UI_Editor/fonts, replace font yang ada.

- Custom text size, opsi ini ada di semua input text.

- Custom text color, opsi ini ada di semua input text.

untuk text nama dan quote, previewnya ada di layout preview, adapun untuk text other, previewnya langsung di kolom input text.

- Semua text input udah support enter keyboard.

- itu aja... selebihnya sama ama v.1.



====Version : 1.0====

==== FITUR ====

- Object selection, karena tidak semua sudut photo ingin kita tampilkan di layout.
Fitur ini ada waktu milih photo buat profile dan cover.

- Crop Circle, buat yg suka photo bulet.

- Layout option, pengen bulet apa kotak? terserah agan deh...

- Save photo profile, barangkali agan perlu photo hasil cropping buat hal lain. Photo akan otomatis disimpan di folder sdCard/UI_Editor. Fitur ini akan aktif setelah agan milih photo di gallery.
Photo yang disimpen ukurannya akan beda-beda tergantung resolusi layar,
di Cross a7s, photo tersimpan dengan ukuran 450 X 450px

- Fitur lainnya cari sendiri ya gan.. perasaan ngga ada lagi soalnya.. hehehe..
